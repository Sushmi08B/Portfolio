## üöÄ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR-GITHUB/bi-portfolio.git
cd bi-portfolio
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run Development Server

```bash
npm run dev
# or
yarn dev
```

Then visit `http://localhost:3000`
